# open-charge-map-india-relational-database

# EV Charging Database Project

## Objective
Design a normalized relational database for EV charging infrastructure in India.

## Dataset
- Charging Stations
- Charging Units

## Steps Performed
1. Raw data ingestion
2. Data cleaning (state normalization, cost extraction, datetime conversion)
3. Schema design (3NF)
4. Table creation
5. Data insertion
6. Foreign key enforcement

## Tables
- stations
- units
- operators
- countries
- connection_types
- status_types
